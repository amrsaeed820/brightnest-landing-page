# BrightNest - Premium Cleaning Services

A modern, responsive, and sunset-inspired landing page designed for a professional house cleaning service business.

## Features
- **Responsive Design:** Looks great on desktops, tablets, and mobile devices.
- **Modern UI/UX:** Clean aesthetics with sunset-inspired color palettes.
- **Structured Layout:** Includes Home, About, Services, Blog, and Contact sections.
- **Contact Form:** Ready-to-use form layout for customer inquiries.

## 🛠️ Tech Stack
- HTML5
- CSS3
- JavaScript (Vanilla)
- [FontAwesome](https://fontawesome.com/) (For UI Icons)
- [Google Fonts](https://fonts.google.com/) (Outfit font family)
